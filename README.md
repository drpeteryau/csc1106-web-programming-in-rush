# CSC1106 – Web Programming in Rust

This repository contains teaching materials, examples, and practical exercises for learning web programming using the Rust programming language.

The project is designed for beginners and students who want to understand modern web development concepts together with Rust fundamentals.

---

# Learning Objectives

By completing this repository, students should be able to:

• Understand basic Rust programming concepts  
• Learn Rust ownership, borrowing, and references  
• Build command line and web applications using Rust  
• Understand modern web technologies and frameworks  
• Develop simple backend web services in Rust  
• Explore safe and high performance systems programming concepts  
• Apply Rust in practical web development scenarios  

---

# Topics Covered

## Rust Fundamentals

• Variables and data types  
• Functions and macros  
• Ownership and borrowing  
• References and mutable references  
• Structs and enums  
• Pattern matching  
• Error handling  
• Modules and packages  

---

## Web Programming Fundamentals

• HTTP basics  
• Client-server architecture  
• RESTful APIs  
• JSON handling  
• Routing  
• Request and response handling  

---

## Rust Web Development

• Rust web frameworks  
• Building web servers  
• API development  
• Form handling  
• Database integration  
• Static file serving  
• Web application structure
